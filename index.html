Feature Scaling in Data Science and Machine Learning Feature scaling is a technique applied to columns of data. There are two main types of feature scaling: Normalization Standardization Normalization Normalization scales all values to a range between 0 and 1. $$ X^{\prime}=\frac{X-X_{\min }}{X_{\max }-X_{\min }}$$ Standardization Standardization scales values to have a mean of 0 and a standard deviation of 1, typically resulting in values between -3 and +3, although values can exceed these limits in some cases.…
